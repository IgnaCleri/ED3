# Imagenes del manual operativo

Esta carpeta queda reservada para figuras, capturas, pinouts y recursos graficos del nuevo manual operativo del LPC1769.

Los recursos que se agreguen aqui deberian:

- provenir del datasheet del microcontrolador o del PDF de la placa de desarrollo;
- quedar recortados de forma legible;
- mantener coherencia visual con el resto del documento;
- usarse solo cuando aporten claridad real a la consulta tecnica.
